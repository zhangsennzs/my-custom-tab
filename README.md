# 🌌 我的自定义标签页 (Geek Tab)

一款美观、轻量且高度可定制的浏览器新标签页扩展 (Manifest V3)。专为热爱折腾网络部署、搭建本地服务以及管理高频资讯的极客玩家打造。无缝接管默认桌面，集成内网直达、分类管理与全自由拖拽布局。

## ✨ 核心特性

* **🗂️ 毛玻璃分类文件夹**：采用响应式双列 Grid 布局与沉浸式毛玻璃 (Glassmorphism) 动效。自由创建“内网与实验室”、“游戏与娱乐”等专属分类，告别桌面杂乱。
* **🖱️ 物理级拖拽重排**：内置本地化 `SortableJS` 引擎。支持跨文件夹拖拽网站卡片，支持分类整体拖拽排序，动画丝滑且纯净，彻底屏蔽了浏览器原生拖拽的“文字选中”干扰。
* **🌐 智能内网路由解析**：重写了搜索框正则逻辑，完美识别 `192.168.1.1` 或 `localhost:8080` 等本地 IP 与端口。告别局域网调试时被误当成搜索词的烦恼，回车即可极速直达旁路由或本地知识库。
* **💾 无限制本地存储**：全面接入 `chrome.storage.local` 并开启 `unlimitedStorage` 权限，彻底突破 5MB 容量瓶颈。极速上传并长久保存原画级高清壁纸，无惧浏览器缓存清理。
* **🛠️ 极客专属小组件**：
  * **自动保存便签**：随时随地记录网络冗余 (HA) 架构的设计灵感或临时的代码片段。
  * **极简待办事项**：规划家庭网络搭建步骤，或追踪对比 Ryzen AI 9 迷你主机等设备的评测清单。
  * **倒数日引擎**：精准追踪重要日程或期待已久的 PC 游戏大作发售倒计时。

## 🚀 安装指南

本项目为原生浏览器扩展，无需上架繁琐的应用商店，可通过“开发者模式”一键极速部署：

1. 点击仓库右上角的 `Code` -> `Download ZIP` 下载代码，并解压到一个固定的本地文件夹中。
2. 打开 Chrome 或 Edge 浏览器，在地址栏输入 `chrome://extensions/`（Edge 为 `edge://extensions/`）并回车。
3. 开启页面右上角的 **“开发者模式”**。
4. 点击左侧的 **“加载已解压的扩展程序”**。
5. 选择你刚刚解压的文件夹（需确保文件夹内包含 `manifest.json`）。
6. 打开一个新标签页 (`Ctrl + T`)，享受你的全新桌面！
